Opticos “Missing Middle” Proposal

Place-type framework from Opticos Design’s Map Analysis, georeferenced and encoded onto all — Oak Park parcels.

Proposal source dated 2026-06-08 · analysis, not final zoning
What this is
Each parcel is tagged with the Opticos Place Type that covers it, derived by georeferencing the Place Types & Transit map (~12 m accuracy) and sampling it under each parcel polygon.

Caveat: this is Opticos’s existing-conditions / framework map — not the final parcel-level proposed zoning (expected around the 2026-07-28 Village Board vote). Corridor = parcels fronting a designated corridor street.

Sources: parcels — Cook County GIS (via oak-park-properties); proposal — Shape Oak Park.
